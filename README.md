# Financial Tools

Privacy-focused, browser-only personal finance calculators for residents of **Ontario, Canada**.

All calculations run entirely in your browser - no data is ever sent to a server.

## Tools

- **Life Insurance Needs Calculator** - Estimate how much life insurance coverage you may need based on your income, debts, dependants, and financial goals.

## Tech Stack

- [Astro](https://astro.build/) v6 - static site generation
- [React](https://react.dev/) v19 - interactive calculator components
- [Tailwind CSS](https://tailwindcss.com/) v4 - utility-first styling
- TypeScript

## Development

```bash
npm install        # Install dependencies
npm run dev        # Start dev server at localhost:4321
npm run build      # Build for production
npm run preview    # Preview production build
```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions on push to `main`.

## Privacy

Every tool runs entirely in the browser. There are no cookies, analytics, tracking scripts, or server-side data processing of any kind.
